STREAMLIT WEB APP

While making my ML projects I found a need for a tool that could instantly show up my output like a web app without having to write huge codes of html,css,javascript so here I am learning to build web apps with streamlit
I followed the following GitHub repository to learn the same
GitHub Repo link : (https://github.com/zkan/30DaysOfStreamlit/tree/main)

## 🚀 Features
LEARNINGS :
- Basic functions like st.write,st,header,st.title,etc
- Using numpy and pandas with streamlit
- Using altair to plot graphs


## 🛠️ Tech Stack
* Framework : Streamlit
* Language : Python
* Core Libraries: Pandas, NumPy, Altair
